# file server

类似于 Nginx 的 autoindex 模块的 HTTP 文件服务器

人总是有一些奇奇怪怪的需求，比如偶尔要在两个环境中传东西，一直想找一个简单又方便的工具总是找不到。不是功能太杂，就是前戏配置麻烦，再要不就属于长久性的，不符合我们临时用用的精神理念

于是就有了这个东西，即下即开即用

现已支持 autoindex, webdav, httpfs 三种运行状态

## 快速上手

下载 Release 中已编译好的程序，直接运行即可，默认监听 8080 端口，并将当前所在目录作为文件服务器根目录

## 使用说明

添加 -h 参数查看帮助信息
